# Iris Flower Classification

## Objective

Classify Iris flowers into three species:

- Setosa
- Versicolor
- Virginica

using Machine Learning.

## Dataset

The Iris dataset is provided by Scikit-Learn.

Features:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

Classes:

- Setosa
- Versicolor
- Virginica

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Steps Performed

1. Loaded the Iris dataset.
2. Explored dataset information.
3. Visualized data using a scatter plot.
4. Split data into training and testing sets.
5. Trained a K-Nearest Neighbors (KNN) classifier.
6. Predicted flower species.
7. Evaluated the model using:
   - Accuracy Score
   - Confusion Matrix
   - Classification Report

## Model Used

K-Nearest Neighbors (KNN)

## Run

```bash
python iris_classification.py
```

## Result  

The model successfully classifies Iris flowers with high accuracy.
